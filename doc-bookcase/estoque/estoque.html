Módulo Estoque

VISÃO GERAL

O módulo de estoque do BookCase foi desenvolvido para facilitar o controle de livros e produtos do sebo. Nele é possível cadastrar itens, atualizar informações, consultar disponibilidade e acompanhar o acervo de forma simples e organizada.

Esta página reúne manuais rápidos e explicações sobre cada tela disponível no módulo.

TELAS DO MÓDULO

Utilize a lista abaixo para acessar rapidamente a documentação de cada tela do sistema.

  • Cadastro de livros
  • Listagem de livros
  • Categorias

CADASTRO DE LIVROS

A tela de cadastro de livros permite adicionar novos itens ao estoque do sistema.

Funções disponíveis:

  • Cadastro de título, autor e editora;
  • Definição de categoria;
  • Inserção de preço;
  • Controle de quantidade;
  • Registro de data de entrada.

Após salvar o livro, ele ficará disponível automaticamente na listagem de estoque.

LISTAGEM DE LIVROS

A listagem de livros mostra todos os itens cadastrados no sistema.

Nesta tela você pode:

  • Pesquisar livros;
  • Consultar disponibilidade;
  • Editar informações;
  • Visualizar preços;
  • Controlar entradas e saídas.

CATEGORIAS

O sistema permite organizar os livros utilizando categorias personalizadas.

Isso facilita pesquisas, organização interna e localização rápida de itens no estoque.

CADASTRO DE PRODUTOS

Além de livros, o BookCase também permite cadastrar outros produtos vendidos pelo sebo.

Exemplos:

  • Marcadores;
  • Quadrinhos;
  • Mangás;
  • Itens colecionáveis;
  • Revistas.

DÚVIDAS COMUNS

O livro desapareceu da listagem. O que fazer?

Verifique se o item foi marcado como indisponível ou removido do estoque.

Posso editar um livro já cadastrado?

Sim. Basta acessar a listagem de livros e selecionar a opção de edição.
